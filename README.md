## Setup Project

1. Create virtual environment and install dependencies:

```bash
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

- OR

```bash
python3 -m venv env
source env/Scripts/activate
pip install -r requirements.txt
```

2. Initialize and Run the project:

```bash
chmod +x setup.sh
./setup.sh
```
